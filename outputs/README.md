# Outputs

This folder receives summary tables and reports generated by running the analysis pipeline: data_quality_report.json, kpi_summary.json, churn_by_contract_type.csv, tenure_summary.csv, monthly_charges_summary.csv, segment_summary.csv, and clv_by_segment.csv. These files are BI-tool ready and can be loaded directly into Power BI, Tableau, or Excel. Run python -m src.main to populate this folder.
