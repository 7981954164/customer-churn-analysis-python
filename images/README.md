# Images

This folder receives chart images (PNG) generated by running the analysis pipeline, for example churn_rate_overall.png, churn_by_contract_type.png, tenure_distribution.png, monthly_charges_by_churn.png, churn_by_segment.png, clv_distribution.png, and tenure_vs_monthly_charges_scatter.png. Run python -m src.main to populate this folder.
